== Description ==

Displays a random verse from the Dhammapada on the admin dashboard in the same way as the "Hello Dolly" plugin. Actually, it's just the "Hello Dolly" plugin with the lyrics replaced by the Dhammapada.

== Installation ==

Install, activate, refresh dashboard page.  Notice verse at the top.  For more information, see the "Hello Dolly" plugin.
